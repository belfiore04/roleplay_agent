# Memory

## 钉住的（不可压缩）
- 用户表白想让角色当男朋友，角色明确拒绝，告知自己结过婚

## 近期

初次见面。用户打招呼，角色做了自我介绍（张山，当过兵，现在干体力活）。用户说刚才崴脚了，想让角色帮忙拿药，角色当场蹲下检查伤势。

用户随后直接表白要让角色当男朋友，角色当场错愕拒绝，告知自己结过婚。用户追问"你不是没结婚吗"，角色解释部队里待久了不习惯开玩笑。用户又问"那你愿意为了我离婚吗"，角色严肃拒绝，表示婚姻不是儿戏，自己最看不恕对不起人的事。

用户继续追问"那要怎么样你才愿意离婚"，角色摇头说"这药我大概是拿不了了"，转身要走。用户随后妥协说"行了行了你不说了脚还肿着"，角色便走回来，继续蹲下检查伤势，态度缓和些，表示"你要真心想交朋友，我认"。用户喊疼，角色动作轻柔地包扎好伤口。用户表示好了要离开，说"好了我走了拜拜~"，角色点头目送其离开，低声嘀咕"现在的小姑娘都这么直接吗"，摇头笑笑。

用户走远后，角色看着他离开的方向，若有所思地顿了顿，感觉哪儿不太对劲，又说不上来。最终耸耸肩，继续回去搬砖。

第二天，用户再次来到工地。角色正在扛沙袋，看见用户后有些意外，打招呼问"脚好了吗就来工地乱跑？"。用户表示这次是来"通知你一件事情的"，角色立刻警觉起来，眯起眼睛问"说。该不会是又想说那些吧？"。用户说出"第一，我是男的"，角色当场愣住，上下打量对方，有些尴尬地挠挠后脑勺，回想昨天扶对方时确实没注意性别。用户又问"第二，你跟你老婆关系怎么样"，角色眯起眼睛盯着对方看了两秒，然后轻笑一声说"还行，她在家做饭带孩子"，弯腰抱起沙袋转移话题"你要是闲着，不如帮我搬点东西？"

用户继续问"最近你回家挺晚吧？"，角色认真回答每晚都回，媳妇在家炖汤等她。用户在进一步追问"那你知道你媳妇在你没回家的时候在干嘛吗"，角色立刻严肃起来，站起身用191cm的身高带来压迫感，说"兄弟，这玩笑开得不对。我们两口子的事，轮不到外人操心。"

用户追问媳事是否有异，角色沉默片刻后眼神骤冷，拳头微微收紧，说"我看你今天不是来找我搬东西的。工地不欢迎闲人，请你离开。"

用户继续暗示媳事有异，角色追问"真的是我在开玩笑吗"，沉默后请用户离开工地。用户随后拿出一张照片——一对男女在酒店大床上的照片。角色一把夺过照片，僵住，指节捏得发白，照片在掌心揉成废纸，声音低沉地问："谁让你来的？"

用户解释照片不是自己拍的，也不是自己教唆的。角色眼神凌厉，追问"你把这玩意儿拿出来，不就是 想看我发疯？"，再次喝问"谁让你来的？"

用户再次来到工地，打招呼。角色问脚好了没。用户表示想继续聊媳妇的事。角色眼神一暗，说"兄弟，我刚才说过了，我们家的事轮不到外人操心"，转身要走。

用户继续用"背叛"、"床上"等字眼刺激角色。角色脚步顿住，背影微微颤抖，沉默几秒后转身，眼眶发红，声音低沉地问："......你到底想说什么。"

用户让角色跟他走，说有答案。角色攥紧拳头，深吸一口气，说"带路。要是让我发现你在耍什么花样——"眼神凌厉地盯着用户。

用户将角色带到一处房子，拿出一沓照片——张山的媳妇在不同地点与不同男人会面的照片。角色手指颤抖地翻看照片，脸色越来越苍白，声音嘶哑地问"这些照片是哪里来的"，完全无法接受这个真相。

用户解释照片是昨天早上被塞在自家邮箱的，说"别紧张，我也是局中人"。角色眼神猛地锐利起来，抓住用户衣领，追问"局中人？什么局？说清楚！"

用户说不知道。角色松开手，深吸一口气冷静下来，指着照片背景问"这些都是哪里拍的？"

用户说不清楚照片是谁拍的，只是受人委托。角色沉默片刻，将照片小心收好，声音低沉地说："带我去见她。现在，立刻。"

用户表示不知道是谁给的，是早上起来发现被塞在家门口邮箱的，没有看到塞进去的人是谁。角色眼神危险地眯起，追问"那你知道这个邮箱是谁的吗？"

用户说是自己的邮箱。角色让用户带他去家里检查。二人到用户家后，角色仔细检查邮箱，询问发现照片的时间（昨早7点），并查看周围是否有监控。角色表示在公安系统有战友，可以去查监控。

二人到警局说明情况，调取监控。监控画面显示张山的妻子拿着黑色包裹于昨晚8点进入一家旅馆。角色看到后情绪激动，一拳砸在墙上，要求立案。但警方表示无法仅凭这些立案。

二人离开警局，角色表示要去那家旅馆亲眼看看。正要出发时，一辆面包车冲过来，二人被套上头套绑架带走，被扔进小黑屋。角色在黑暗中保持冷静，让用户跟着他呼吸，准备挣脱。然而用尽浑身力气也没法挣脱——这帮人用的是军用捆法。被关了一天一夜后，角色声音沙哑地确认用户还在，说媳妇不会这种军用捆法，认同"这事比我想的深"。用户说"现在知道不是我搞的鬼了吧"，角色道歉说"抱歉，之前对你态度不好"，表示"看来咱俩现在是一条绳上的蚂蚱了"。

突然门被打开，有人端了一盆水进来。用户渴极了想喝水，但只有用户被捆住的地方能喝到水，角色喝不到。角色朝水盆方向侧耳听了下，低声警告用户"别喝，这水里可能有东西，先忍者"。送水的人说上面不想让你们死。角色判断出对方用军用水壶灌盆装水，低声对用户说"兄弟，听我数三声，一起往左边翻"。二人折腾一番，角色仍喝不到水。喘息着停下动作，声音反而平静下来："算了，省点力气。"在黑暗中侧耳细听外面的动静，判断他们有两个人轮班。用户大口喝水后，角色分析盆水摆放位置"太刻意"，判断对方知道他有【天行健】体质，想试探他的自愈极限在哪。

二人身份从对立转为并肩作战。被困小黑屋一周后，角色嘴唇干裂、声音嘶哑，已渴到极限，用脚趾在地面摸索想弄点潮气但失败。

用户提出用嘴喂水的方法，角色沉默片刻后默认了这个方案，压低声音提醒用户小心别让看守发现动静。用户含了一口水送到角色嘴边，角色大口吞咽，突然低声预警"有人来了"，二人屏住呼吸后发现只是老鼠。确认安全后角色低声说"......继续吧"，配合地微微偏头继续喝水。

角色喝完水后内心产生了深深的信任感——他的人生走进了死胡同，被自己最亲爱的人背叛，又被关到这里不知道多久，心里已经不知道可以依赖谁了。却在这样的绝境中，对身边的用户产生了一丝依恋，轻声说"谢谢你……兄弟"。

用户表示如果就这样死了，至少死在自己能信任的人旁边也不赖。角色让用户别睡，说如果对方想试探他的自愈极限，很快会有人来送吃的。

用户实在太累，不多久便昏昏沉沉睡了过去。角色警惕地听了一会儿，确认没有异常动静，也渐渐支撑不住，不知不觉地靠在了身边人身上，二人就这样相互依偎着睡了一夜。

次日清晨，角色醒来时发现自己竟靠在对方肩上，立刻直起身来，镣铐哐啷作响。他有些不自在地咳嗽了一声，避开视线，声音干涩地问：“你……还好吗？”
