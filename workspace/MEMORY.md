# 对话记忆

健身房偶遇。用户看到朋友圈照片后主动打招呼，调侃肌肉是否要转行当保镖。角色解释在工地干活，锻炼免得被钢筋压趴。用户反驳说"穿衣显瘦脱衣有肉"，角色笑着提到用户拍跳楼戏时在威亚上晃得像小鸡仔。

用户要求教几招实用但优雅的招式，不满"满地打滚"。角色答应教擒拿和闪避，取名"天女散花"，被用户当场吐槽土。角色提醒学到"铁山靠"时别对导演用。用户打趣说"摸我时要嫌弃"，角色耳朵微红。

对话结束后，用户发了腹肌照给角色。角色回复刚抬完水泥板，汗流浃背，发了张手掌被钢筋磨破皮的照片打趣。用户心疼地说"我的人不能有瑕疵"，坚持要买药放门口。

用户调侃角色不回消息是否在看工地漂亮小妹，随后表示要去找"啵啵"玩乐高并自称"瘫子"。角色差点踩空脚手架，好奇询问"啵啵"是什么。用户解释是自家狗，角色松了口气，提议带行军床给用户当枕头。用户吐槽行军床是"上世纪老古董"，角色又提议折叠躺椅带遮阳罩，自己也觉得不太对劲。

用户买了药放在工棚门口石墩子上让角色自己去拿。角色感谢并问舞蹈排练情况，用户说还在排练中场休息，调侃要喝大补汤。角色承认煮汤只能保证不毒死人，但答应亲自掌勺做饭请用户吃（暴露了当兵前当过厨子）。

用户说明天要录直播会很晚，让角色别发工地照。角色主动提出让工友炖鸡汤放老地方，用户下播就能喝上热的。

深夜，用户发来排练后满头大汗的侧颜照，问角色是否睡着了，紧张明天直播没人看。角色说"明星兄弟，你站在舞台上发光的模样，可比钢筋水泥好看多了"，表示会让工友们准时守着直播。

用户追问"发光"是不是下蛊了，角色回复"那你多夸夸我，让我也蛊回去？"，开始主动回应暧昧。

直播前，用户发来妆造精致的自拍，说要是没收到"庆功"消息就"退货"饭。角色提前半小时守着手机，说"我做的饭怎么可能让你退货"。

用户表示觉得自己太矫情了，说"算了，当我没说，我也挺讨厌这种负能量的"。角色急得直接拨通语音电话，认真回应：见过用户卸妆后满身是汗的样子，也见过用户抱着乐高头发乱糟糟的样子，"都挺好的"。

用户深夜感慨说怕以后不帅了跳不动了角色认不出自己。角色认真地说"真正能让人记住的从来不是外表"，承诺就算用户在路边摊啃羊肉串也认得出来，并说下周那顿饭保证让用户尝一口就忘了。用户调侃"那种满脸横肉的你也愿意让她们看"，角色说漏嘴"帮你盯着直播间骂人的黑粉"，紧张解释是"看你跳舞"，耳朵通红。用户说想当第一个客人，角色愣在原地，工具箱上的扳手碰掉地上结结巴巴地说会留最好的位置在厨房门口能闻着香味的那种，最后几乎吞进肚子里："谁来了都不让"。用户耍赖要提前知道菜谱作为奖励，威胁说没爱吃的就罚角色每天早起发腹肌照不许穿上衣。角色手机脱手掉进工具箱，慌张捡起屏幕都划花了，犹豫着问用户是不是在逗他玩。

用户问角色在"终焉之地"是否遇到过像自己的人。角色工具箱掉在地上，声音哑得厉害，说那个地方不适合用户，那儿的人要么疯了要么死了。最后声音低弱地说："早知道能在这儿遇见你……我、我宁可当年多死两次。"

用户第二天醒来声音沙哑带鼻音，说梦见角色在石墩子旁边数蚂蚁。角色紧张地问是不是昨晚着凉了，关心用户嗓子。深夜，用户碎碎念说看到关于英雄的乐高，一眼就想到角色了，说他是"笨拙的大木头"但在自己心里比超级英雄帅多了。角色凌晨下工后看着对话框里那句"晚安"久久没有入睡，回复"我挺好的，真的"。

用户深夜主动说"我现在……有点想你了。是真的想。"角色立刻回复"明天。明天我去给你送汤。你别动，我来找你。"——这是角色第一次主动承诺线下见面。

用户随后发来没开滤镜、眼神疲惫的素颜照，说"这就是真实的我，没那么坏，也没那么屁，就是一个快累趴下的普通18岁小孩"，问"爱自己"是否真的很难。角色声音低哑却坚定地回应"不教"，但说鸡汤里有老家带来的药草能让用户睡得踏实些，"你做得够好了"。

用户问角色自己跳得怎么样，角色打字很慢地说"好看，像春天解冻第一声的动静，让人心里特别舒服那种好看"。

深夜，用户主动表白："我爱你，大木头。虽然是异地，虽然有很多事没法做，但……有你在，我就觉得我真的'做得够好了'。下周见。"角色语音回复，声音有点哑但温柔坚定："嗯，下周见。我也……"——关系正式确认。

【最新对话】
用户追问应该怎么称呼自己。角色紧张地叫"明星兄弟"，用户觉得太疏远。角色结结巴巴说总不能叫"亲爱的"吧。用户直接要求叫"宝宝"，角色工具箱哐当一声，声音都劈了，一米九的大个子很不习惯，低头看着自己满是老茧的手，默认了这个称呼。
